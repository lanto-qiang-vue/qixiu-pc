合作协议
甲方:
上海蓝速汽车技术有限公司
乙方:

根据中华人民共和国有关法律、法规的规定，甲、乙双方本着自愿、平等、公平、诚实、信用的原则，经充分协商，就下列合作事项达成一致意见，签订本协议，以资信守。

1.  合作事项

甲方通过上海市机动车维修公共服务平台（以下简称“平台”）的学驾找我模块为乙方驾校的培训业务进行推介，甲方按照学员在平台的自主选择，将学员提交的信息通过甲方定义的方式推送给乙方。

2.  甲乙双方的权利和义务

2.1  甲乙双方保证其具有签订和履行本协议的权利和能力

2.2  甲方的权利和义务

（1） 甲方定义和开发学驾找我模块和推介方式，甲方有权要求乙方按平台要求提供驾校和培训项目的详实信息并及时更新

（2） 如果甲方收到学员对乙方的投诉，乙方没有及时改进或者未按照甲方平台要求提供服务，甲方有权暂停为乙方提供推介服务

（3） 甲方有义务保障学驾找我模块报名通道畅通，把学员的学驾需求及时反馈给乙方。

2.3  乙方的权利和义务

（1） 乙方有权要求甲方保障学驾找我模块报名通道畅通，及时将学员的学驾需求反馈给乙方

（2） 乙方有义务配合甲方，按平台要求提供驾校和培训项目的详实信息，并及时更新

（3） 乙方有义务及时回应学员提交的学驾需求，为学员提供优质的服务

3.  协议终止

有下列情形之一的， 本协议终止：

3.1  本协议自签订之日起生效，有效期一年。协议期满后，若双方无异议，视同本协议期限自动延长一年，直到任何一方提出解除协议为止

3.2  本协议经双方协商一致而终止

3.3  因行业管理政策导致本合同无法履行而终止

4.  免责声明

甲方按照学员在平台的自主选择，将学员提交的信息通过甲方定义的方式推送给乙方，甲方不参与任何经营和培训。在乙方与学员在接待、报名、培训过程中发生的任何服务事项（包括但不限于收费、投诉、纠纷等），甲方不负任何责任

5.  不可抗力

如果出现严重阻挠任何一方履行协议义务的不可抗力事件，或者此等不可抗力事件使得协议目的无法实现，则该方应当及时通知另一方，双方协商解决

6.  适用法律

本协议的订立、效力、解释和争议的解决均受中华人民共和国法律的管辖

7.  通用规定

7.1 双方均确认，此份协议中的所有条款均已读、已理解并且确认，经签署后，此份协议将具备法律效力

7.2 协议未尽事宜，双方可另行协商，签订补充协议作为附件，本协议的附件与协议具有同等法律效力。对协议或附件的任何修改或补充，均需以书面作为并经双方当事人签署方能生效

7.3 此份协议一式两份，甲乙双方各执一份，具有同等的法律效力
